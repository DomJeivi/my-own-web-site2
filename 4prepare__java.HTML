Java o que é?

Java é uma linguagem de programação de propósito geral conhecida por sua portabilidade e capacidade de escrever programas de diversos tipos e funcionalidades, além de executá-los em várias plataformas diferentes.

Onde e por que aprender?

O Codecademy oferece um curso interativo de Java que é adequado para iniciantes ele cobre os conceitos básicos da linguagem e permite que você pratique escrevendo código diretamente no navegador, além da Codecademy também há a Coursera , por onde várias universidades oferecem cursos de Java .
