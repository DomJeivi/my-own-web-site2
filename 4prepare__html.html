HTML 5 o que é?

HTML 5 é uma linguagem de marcação para produção de páginas WEB modelo Java HTML para sua formatação inicial e usa documentos CSS para sua personalização.

Onde e por que aprender?

O HTML é uma linguagem simples de ser aprendida e para isso pode aprender gratuitamente pelo w3schools, quem trabalha com HTML normalmente trabalha com web desing podendo ser um serviço de freelancer ou CLT em certas empresas.
